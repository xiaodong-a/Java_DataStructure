### 1、二维数组的查找

​		在一个二维数组中（每个一维数组的长度相同），每一行都按照从左到右递增的顺序排序，每一列都按照从上到下递增的顺序排序。请完成一个函数，输入这样的一个二维数组和一个整数，判断数组中是否含有该整数。

```java

//1、从左下角开始查找
public class Solution {
    public boolean Find(int target, int [][] array) {
		if(array == null || array.length == 0 || array[0] == null) return false;
        int rows = array.length;
        int cols = array[0].length;
        int r = rows - 1;
        int c = 0;
        while(r>=0 && c < cols){
            if(array[r][c] > target){
                r --;
            }else if(array[r][c] < target){
                c++;
            }else{
                return true;
            }
        }
        return false;
    }
}
```

### 2、替换空格

​		请实现一个函数，将一个字符串中的每个空格替换成“%20”。例如，当字符串为We Are Happy.则经过替换之后的字符串为We%20Are%20Happy。

```java
//如果就在原来的数组上做替换，从后往前，较少的移动，但是对于java的StringBuffer类型就有些奇怪，即使底层用数组，空间为100，但是如果只放入10个数，也操作不了charAt(99)
public class Solution {
    public String replaceSpace(StringBuffer str) {
        StringBuilder sb = new StringBuilder();
        char c;
        for(int i=0;i<str.length();i++){
            if((c = str.charAt(i))==' '){
                sb.append("%20");
            }else{
                sb.append(c);
            }
        }
         return sb.toString();
    }
}
```

### 3、从尾到头打印链表

​		输入一个链表，按链表从尾到头的顺序返回一个ArrayList。

```java
/**
*    public class ListNode {
*        int val;
*        ListNode next = null;
*
*        ListNode(int val) {
*            this.val = val;
*        }
*    }
*
*/

//1、很容易想到ArrayList类的一个方法，add(0,val)，每次向0位置添加
import java.util.ArrayList;
public class Solution {
    public ArrayList<Integer> printListFromTailToHead(ListNode listNode) {
        ArrayList<Integer> list = new ArrayList<Integer>();
        while(listNode != null){
            list.add(0,listNode.val);
            listNode = listNode.next;
        }
        return list;
    }
}


//2、利用栈结构，一种方法是递归
import java.util.ArrayList;
public class Solution {
    ArrayList<Integer> list = new ArrayList<Integer>();
    public ArrayList<Integer> printListFromTailToHead(ListNode listNode) {
        if(listNode != null){
            printListFromTailToHead(listNode.next);
            list.add(listNode.val);
        }
        return list;
    }
}

//3、利用栈结构，使用Stack
import java.util.ArrayList;
import java.util.Stack;
public class Solution {
    public ArrayList<Integer> printListFromTailToHead(ListNode listNode) {
        Stack<Integer> stack = new Stack<Integer>();
        ArrayList<Integer> list = new ArrayList<Integer>();
        while(listNode != null){
            stack.add(listNode.val);
            listNode = listNode.next;
        }
        while(!stack.isEmpty()){
            list.add(stack.pop());
        }
        return list;
    }
}

//4、反转链表
import java.util.ArrayList;
import java.util.Stack;
public class Solution {
    public ArrayList<Integer> printListFromTailToHead(ListNode listNode) {   
        ArrayList<Integer> list = new ArrayList<Integer>();
        ListNode re = reverse(listNode);
        ListNode head = re;
        while(re != null){
            list.add(re.val);
            re = re.next;
        }  
        reverse(head);  //恢复链表
        return list;
    }
   public ListNode reverse(ListNode head){
    ListNode pre = null;
    ListNode next = head;
    while (next!=null){
      next = head.next;
      head.next = pre;
      pre  = head;
      head = next;
    }
    return pre;
  }
}
```
